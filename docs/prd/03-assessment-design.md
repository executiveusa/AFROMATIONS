# Assessment Design (Phase 0)

## Layers
1. Passive metrics: consistency, cadence, hesitation, completion.
2. Active recall: explain in own words, distinction checks.
3. Spoken performance: pronunciation, fluency, correction responsiveness.
4. Vision integrity mode: optional, explicit consent only, adult-only.

## Integrity Safeguards
- Consent artifact required before camera-based integrity checks.
- No covert collection.
- Operator-visible audit trail for every integrity flag.

## Gate Policy
- No unlock without mastery.
- No advanced arc without retained prerequisites.
