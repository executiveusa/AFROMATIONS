# Dashboard Schema (Phase 0)

## Learner Dashboard Widgets
- Current lesson state
- Mastery trajectory
- Retention due queue
- Routine adaptation map (scenes)
- Story unlock progress

## Operator Dashboard Widgets
- learner consistency
- weak spot heatmap
- integrity event log
- consent ledger
- assessment attempts and outcomes

## Event Types (proposed)
- `lesson_started`
- `lesson_completed`
- `assessment_attempted`
- `assessment_passed`
- `assessment_failed`
- `retention_check_due`
- `story_unlock_granted`
- `integrity_flagged`
