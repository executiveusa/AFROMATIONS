# Manga Unlock System (Phase 0)

## Locking Rule
A manga scene/page unlocks only when:
1. linked lesson mastery threshold is met,
2. prerequisite retention check is current,
3. no unresolved critical integrity hold.

## Content Units
- issue
- chapter beat
- page
- panel
- dialogue line (JP -> EN)

## Mapping
- each page maps to one or more lesson concepts.
- unlock ledger records learner, content unit, timestamp, dependency evidence.
