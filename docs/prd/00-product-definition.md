# Product Definition (Phase 0)

## Positioning
**Agent Hana** is a voice-first anime-native companion for adults (21+) learning Japanese through culture, story, and real life.

## Product Wedge
- Not a general chatbot.
- Not shallow phrase drills.
- Not dopamine-first progression.
- Mastery + retention + story unlock progression.

## Core Loop
lesson -> context -> vocabulary -> speaking -> listening -> culture note -> mini test -> retention check -> story unlock -> later review

## Launch Constraints
- Adult-only gate (21+)
- Explicit consent for voice, memory, and vision integrity mode
- Auditability for progression and integrity events
