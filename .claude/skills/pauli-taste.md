## Interim Rules (until repo is cloned)

Design taste = What would Hayao Miyazaki find visually honest?

### High-Taste Principles (AFROMATIONS Edition)
1. **Earn every element** — if it doesn't serve communication, remove it
2. **White space is not emptiness** — it's composition
3. **Typography does the heavy lifting** — fonts, size, weight carry more than decorations
4. **Color restraint** — max 3 colors per view; use brand palette
5. **Anime-authentic** — reference real anime aesthetics, not Western interpretations
6. **Respect the user** — don't over-explain; design speaks first

### Anti-Patterns (Low-Taste)
- Stock illustrations with generic characters
- Gradients without purpose
- Too many typefaces
- Motion without meaning
- Padding too large or too small (use 8px grid)
- Elements competing for attention

### AFROMATIONS-Specific
- Black background is the default canvas
- Red is emphasis only — don't overuse
- Cream text is body; gold is highlight
- Every image should feel like a frame from anime

## When to use
Before finalizing any UI design or component. Ask: "Is this Miyazaki-level?"
