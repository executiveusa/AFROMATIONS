## Interim Rules (until repo is cloned)

### Content Types
- Anime culture deep-dives (Japanese-first perspective)
- Japanese language guides (grammar, vocabulary, particles)
- Folklore + mythology explanations (kami, yōkai, yūrei)
- Character design breakdowns (AFROMATIONS studio work)
- Learning OS updates (Hana progress, unlocks)

### Voice & Tone
- Direct, respectful, assumes intelligent reader
- Japanese terms introduced properly (term + reading + meaning)
- No hype words, no filler
- First sentence = the point (not "In this article, we'll explore...")

### Post Structure
1. **Headline**: specific, not clever
2. **First paragraph**: the core insight, immediately
3. **Sections**: each with a distinct point (not "background")
4. **Japanese terms**: shown in kanji first, then reading, then meaning
5. **Images**: every 500 words; prefer original art over stock
6. **CTA**: one, at end, relevant to content

### SEO (Simple)
- Primary keyword in headline and first paragraph
- H2/H3 use natural language, not keyword-stuffed
- Meta description = first paragraph, max 160 chars

## When to use
Before generating any blog post. Apply this skill to set voice and structure.
